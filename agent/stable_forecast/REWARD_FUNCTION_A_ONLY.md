# Reward Function A (Khusus Existing Function)
Referensi kode: `environment/environment.py:15`
## 1) Rumus Fungsi A
```python
if response_time <= 50:
    rt_penalty = 0.0
elif response_time <= 80:
    rt_penalty = (response_time - 50) / 30
else:
    rt_penalty = 1.0 + ((response_time - 80) / 80)

cost_raw = action / 99.0

if response_time > 80:
    cost_mult = 0.0
elif response_time <= 50:
    cost_mult = 1.0
else:
    cost_mult = 1.0 - rt_penalty

cost_eff = cost_raw * cost_mult
total_penalty = (w_rt * rt_penalty) + (w_cost * cost_eff)
reward = 1.0 - total_penalty
```
Asumsi tabel di bawah: `w_rt = 1.0`, `w_cost = 1.0`, pembulatan 4 desimal.
## 2) Kondisi Reward Maksimum, Minimum, dan Nol
- Reward maksimum: `1.0000` pada `response_time <= 50` dan `action = 0`
- Reward minimum (teoritis): tidak berbatas bawah (`-> -inf`) saat `response_time -> inf`
- Reward minimum pada rentang uji (`response_time <= 300`): `-2.7500` pada `response_time = 300`
- Reward `= 0`:
  - `action = 99` dan `response_time <= 80`, atau
  - `response_time = 80` untuk action apa pun
